## API documentation
|  parameter  |  illustrate  |  type  |  Defaults  |
|  ---   |  ---  |  ---  |  ---  |
|  isLoading  |  Load the state, default to TRUE  |  bool  |  |
|  loadingText  |  Text of loading state  |  string  |  |
|  hiddenText  |  Whether to hide the text of the loading state  |  bool  |  |
|  type  |  SPIN type, currently there are two types of Ball and LINE  |  string  |  |
|  bgColor  |  Load animation color  |  string  |  |
