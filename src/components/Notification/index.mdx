## Use an introduction
``` js
import { Notification } from '@otter13/vibe-ui'

// use
xNotification.pop({
    type: 'warning',
    message: 'Fun talk about the front -end learning check -in',
    duration: 0,
    description: 'The front -end foundation, the intermediate advance'
})

// Global configuration
xNotification.config({
    placement: 'bottomRight'
})
```

 ## API documentation

 ### xNotification Global attribute
|  Parameter | Description | Type | Default value |
|  ---   |  ---  |  ---  |  ---  |
|  bottom  |  When the message pops up from the bottom, the distance from the bottom, the unit pixel  |  number  |  '24px'  |
|  duration  |  Automatically close the delay in default, the unit seconds  |  number  |  4.5s  |
|  getContainer  |  HTMLNode Configure the output location of the rendering node  |  () => HTMLNode  |  () => document.body  |
|  placement  |  Popping position(topLeft/topRight/bottomLeft/bottomRight)  |  string  |   'topRight'  |
|  top  |  When the message pops up from the top, the position of the top, the unit pixel  |  number  |  '24px'  |
|  closeIcon  |  Customly closed icon  |  HTMLNode  |    |
