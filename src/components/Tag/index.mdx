## API documentation
|  parameter  |  illustrate  |  type  |  Defaults  |
|  ---   |  ---  |  ---  |  ---  |
|  closable  |  Whether it can be closed  |  bool  |  false  |
|  onClose  |  The callback when the label is closed  |  func  |   |
|  color  |  Colors of label,No setting is the default color  |  string  |  '#fafafa'  |
