## API documentation
|  Parameter | Description | Type | Default value |
|  ---   |  ---  |  ---  |  ---  |
|  onClick  |  Clicks of exposed to the outside world  |  func  |  |
|  checked  |  Whether it is selected  |  bool  |  false  |
|  disabled  |  Whether  |  bool  |  false  |
|  offText  |  Turn off the text  |  string  |  ''  |
|  onText  |  Text of the state  |  string  |  ''  |
|  size  |  The size of the component  |  string  |  'default'  |
