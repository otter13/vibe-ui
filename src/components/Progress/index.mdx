# Progress progress bar

#### Basic use
<Playground>
    <Progress percent={50} /><br />
    <Progress percent={20} themeColor="#888" /><br />
    <Progress percent={10} themeColor="red" /><br />
    <Progress percent={88} themeColor="green" />
</Playground>

#### Hidden progress text
<Playground>
    <Progress percent={20} themeColor="#888" /><br />
    <Progress percent={50} hiddenText />
</Playground>

#### Set the progress control width
<Playground>
    <Progress percent={20} width={200} hiddenText /><br />
    <Progress percent={20} width={400} hiddenText themeColor="red" /><br />
    <Progress percent={40} width={300} hiddenText themeColor="green" />
</Playground>

#### Set the progress state threshold
Different progress strips are displayed according to different progress. This case is divided into 20, 50ï¼Œ 80

<Playground>
    <Progress percent={19} statusScope={[[20, 'red'], [50, 'orange'], [80, '#06f']]} /><br />
    <Progress percent={40} statusScope={[[20, 'red'], [50, 'orange'], [80, '#06f']]} /><br />
    <Progress percent={90} statusScope={[[20, 'red'], [50, 'orange'], [80, '#06f']]} />
</Playground>

## usage

``` js
import { Progress } from '@otter13/vibe-ui'

// Import
<Progress percent={50} />
<Progress percent={40} width={300} hiddenText themeColor="green" />
<Progress percent={90} statusScope={[[20, 'red'], [50, 'orange'], [80, '#06f']]} />
```

## API documentation
| Parameters | Description | Type | Default value |
|  ---   |  ---  |  ---  |  ---  |
|  themeColor  |  The color of the progress bar  |  string  |  '#06f'  |
|  percent  |  Progress value percentage  |  number  |    |
|  autoHidden  |  Whether progress reaches 100%Automatically disappear  |  bool  |  false  |
|  hiddenText  |  Whether to hide the progress entry  |  bool  |  false  |
|  width  |  The width of the progress bar  |  string|number  |  320 |
|  textColor  |  Progressive text color  |  string  |  '#666'  |
|  statusScope  |  Status threshold,Set the color bar of the scope of different progress, respectively,Maximum allow setting 3 values, For a two -dimensional array  |  array  |    |
